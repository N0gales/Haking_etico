2. Buscar archivos por nombre  
   Una vez dentro, puedes buscar un archivo cuyo nombre conozcas usando:
   find ~ -name readme

   - 'find' busca archivos y directorios.
   - '~' indica que la búsqueda empieza en tu carpeta home.
   - '-name readme' filtra archivos que coincidan exactamente con ese nombre.

3. Leer el contenido del archivo  
   Para ver qué hay dentro del archivo encontrado:
   cat /home/bandit0/readme

   - 'cat' muestra el contenido de un archivo en la terminal.
   - Necesitas la ruta completa o relativa al archivo.

*Alternativas y consejos:

- En lugar de 'cat', si el archivo es muy largo, puedes usar 'less' o 'more' para leerlo página a página:
  less /home/bandit0/readme
- Si no recuerdas el nombre exacto del archivo, puedes usar 'ls' para listar archivos en un directorio:
  ls -la ~
- Experimenta con 'find' para buscar archivos con criterios distintos (por ejemplo, buscando por extensión o patrón de nombre).
